Detailed Description

template<typename Value_T>
struct regex_table::MutableStateMachine< Value_T >

A Simple class for building your own custom state machines.

it exposes a simple builder pattern to construct these state machines

the MutableStateMachine<void> specialization represents a raw Regular Expression state machine, whereas any other specializations represent State machines with contained values
